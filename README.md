# Price

Статичний каталог товарів для публікації на GitHub Pages.

## Що відкривається

- Головна сторінка сайту: `index.html`
- Вона автоматично відкриває каталог: `markers-demo.html`

## Як викласти на GitHub Pages

1. Завантаж у репозиторій увесь вміст папки `E:\ChatGPT\Price`.
2. На GitHub відкрий `Settings -> Pages`.
3. У `Build and deployment` вибери:
   - `Source` -> `Deploy from a branch`
   - `Branch` -> `main`
   - `Folder` -> `/ (root)`
4. Збережи налаштування.

Через кілька хвилин сайт буде доступний за адресою:

`https://linequantum-svg.github.io/Price/`

## Як оновлювати сайт

1. Змінюєш файли локально в `E:\ChatGPT\Price`
2. Завантажуєш оновлені файли в GitHub
3. GitHub Pages автоматично оновлює сайт

## Важливо

- Файл `.nojekyll` уже доданий, тому GitHub Pages віддаватиме статичні файли без зайвих обмежень.
- Усі картинки, стилі та скрипти використовуються локально з цієї ж папки.
